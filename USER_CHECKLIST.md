## 阶段 1：Keil 工程配置 (预计 10 分钟)

### 1.1 打开工程
- [ ] 双击 `MDK-ARM/STM32F407_CAN_CMD.uvprojx`
- [ ] Keil 成功打开工程

### 1.2 添加源文件
- [ ] 右键 `Application/User` 组
- [ ] 选择 `Add Existing Files to Group`
- [ ] 添加以下 7 个文件：
  - [ ] `../Src/robot_config.c`
  - [ ] `../Src/robot_kinematics.c`
  - [ ] `../Src/robot_control.c`
  - [ ] `../Src/robot_cmd.c`
  - [ ] `../Src/robot_sequence.c`
  - [ ] `../Src/robot_examples.c`
  - [ ] `../Src/usart.c`

### 1.3 编译器配置
- [ ] 菜单：`Project` → `Options for Target`
- [ ] `Target` 标签页：
  - [ ] 勾选 `Use MicroLIB`
- [ ] `C/C++` 标签页：
  - [ ] Floating Point Hardware 选择 `Use FPU`
  - [ ] Language / Code Generation 勾选 `C99 Mode`

### 1.4 编译测试
- [ ] 菜单：`Project` → `Rebuild all target files` (F7)
- [ ] 编译输出显示：`0 Error(s), X Warning(s)`
- [ ] Flash 占用 < 100 KB
- [ ] RAM 占用 < 10 KB

**遇到错误？查看** `BUILD_GUIDE.md` **第 "常见编译错误" 章节**

## 阶段 2：硬件连接 (预计 15 分钟)

### 2.1 CAN 总线接线
- [ ] STM32 PB8 → CAN 收发器 RXD
- [ ] STM32 PB9 → CAN 收发器 TXD
- [ ] CAN 收发器 CANH → 驱动器 1 CANH → ... → 驱动器 6 CANH
- [ ] CAN 收发器 CANL → 驱动器 1 CANL → ... → 驱动器 6 CANL
- [ ] 终端电阻 120Ω 接在总线首端
- [ ] 终端电阻 120Ω 接在总线末端
- [ ] 所有设备共地（GND 连接）

### 2.2 串口接线
- [ ] STM32 PA9 (TX) → USB转串口 RX
- [ ] STM32 PA10 (RX) → USB转串口 TX
- [ ] STM32 GND → USB转串口 GND

### 2.3 电源接线
- [ ] 驱动器电源：24V（推荐）或 12-48V
- [ ] STM32 电源：USB 5V 或外部 5V
- [ ] 确认电源极性正确（⚠️ 防止反接）

### 2.4 驱动器配置
- [ ] 驱动器 1：地址设为 **1**，波特率 **500 kbps**
- [ ] 驱动器 2：地址设为 **2**，波特率 **500 kbps**
- [ ] 驱动器 3：地址设为 **3**，波特率 **500 kbps**
- [ ] 驱动器 4：地址设为 **4**，波特率 **500 kbps**
- [ ] 驱动器 5：地址设为 **5**，波特率 **500 kbps**
- [ ] 驱动器 6：地址设为 **6**，波特率 **500 kbps**
- [ ] 所有驱动器：细分设为 **16**

**配置方法**：驱动器小屏幕菜单 → Settings → CAN
