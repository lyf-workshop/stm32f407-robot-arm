# 触摸控制功能使用指南

## 硬件连接

2.8寸LCD屏幕的触摸控制器（XPT2046）通过软件SPI连接到STM32：

| 触摸信号 | STM32引脚 | 功能 |
|---------|-----------|------|
| T_CLK   | PD13      | SPI时钟 |
| T_CS    | PE0       | 片选 |
| T_MOSI  | PE2       | 数据输入 |
| T_MISO  | PE3       | 数据输出 |
| T_PEN   | 未使用    | 触摸中断（使用轮询模式）|

## 触摸UI界面

烧录固件后，LCD显示触摸按钮界面：

```
┌─────────────────────────────────────┐
│ [J1-] [J1+] [J2-] [J2+]            │  ← 关节1、2控制
│ [J3-] [J3+] [J4-] [J4+]            │  ← 关节3、4控制
│ [J5-] [J5+] [J6-] [J6+]            │  ← 关节5、6控制
│ [ZERO] [STOP] [STATUS] [---]       │  ← 常用功能按钮
│ （预留区域，显示实时状态）           │
│ [1deg] [5deg] [10deg] [30deg]      │  ← 底部：步长选择
└─────────────────────────────────────┘
```

### 按钮功能说明

| 按钮 | 功能 | 颜色 |
|------|------|------|
| `J1+` / `J1-` | 关节1 正转/反转（步长由底部选择）| 绿色/红色 |
| `J2+` ~ `J6-` | 关节2~6 控制 | 同上 |
| `ZERO` | 将当前位置设为零点 | 橙色 |
| `STOP` | 紧急停止所有电机 | 橙色 |
| `STATUS` | 从编码器同步真实角度 | 橙色 |
| `1deg` / `5deg` / `10deg` / `30deg` | 选择转动步长（点击切换）| 灰色/黄色高亮 |

### 使用流程

1. **启动系统**：LCD自动显示触摸UI
2. **选择步长**：点击底部的 `5deg`（默认）或其他步长按钮
3. **点动控制**：点击 `J1+` 使关节1正转5度，点击 `J1-` 反转5度
4. **归零**：移动到初始位置后，点击 `ZERO` 设为零点
5. **同步角度**：如果怀疑角度偏差，点击 `STATUS` 从编码器读取真实位置

## 触摸校准

如果触摸位置不准（点击按钮没反应或错位），需要进行4点校准：

### 校准步骤

1. **启动校准**：通过串口发送命令：
   ```
   touch_calib
   ```

2. **依次点击4个十字**：LCD会显示4个红色十字，按顺序点击：
   - 第1个：左上角
   - 第2个：右上角
   - 第3个：左下角
   - 第4个：右下角

3. **校准完成**：显示 "Calibration Complete!"，触摸屏幕任意位置返回控制界面

4. **验证**：触摸按钮测试是否准确

### 串口命令

```
touch_calib   # 开始触摸校准（需要在LCD屏幕上操作）
```

## 性能参数

| 参数 | 值 |
|------|-----|
| 触摸轮询周期 | 50ms |
| 触摸防抖 | 上升沿检测（按下时触发一次）|
| SPI速度 | ~500kHz（软件SPI）|
| 触摸精度 | 12-bit ADC（0~4095）|
| LCD坐标 | 320×240 像素 |

## 注意事项

1. **首次使用建议先校准**：默认校准参数是通用值，不同LCD屏幕可能有偏差
2. **触摸响应速度**：由于软件SPI + 中值滤波，单次触摸读取约1ms
3. **按钮尺寸**：每个按钮 80×30 像素，适合手指点击
4. **步长选择**：小步长（1°）用于精细调整，大步长（30°）用于快速移动
5. **并发控制**：触摸和串口命令可同时使用，互不干扰

## 故障排查

### 触摸无响应
- 检查触摸排线是否接触良好
- 运行 `touch_calib` 重新校准
- 串口发送 `status` 确认主程序是否卡死

### 触摸位置偏移
- 执行触摸校准 `touch_calib`
- 确认LCD方向（代码中设定为横屏320×240）

### 按钮误触发
- 减小轮询频率（修改 `main.c` 中的50ms改为100ms）
- 增加防抖时间（修改 `touch_calib.c` 中的300ms）

## 扩展功能（可选）

如果需要更复杂的触摸交互，可以扩展：

1. **滑块控件**：拖动滑块连续调节角度
2. **虚拟键盘**：输入精确的角度值或坐标
3. **示教模式**：触摸 "记录" 按钮保存关键帧
4. **轨迹回放**：触摸 "播放" 按钮重现示教轨迹

## 技术细节

- **XPT2046协议**：8位SPI，命令字 0xD0(X) / 0x90(Y)，返回12位ADC
- **中值滤波**：每次读取3次取中位数，降低噪声
- **坐标变换**：仿射变换矩阵（6参数），从原始ADC映射到LCD像素坐标
- **状态机**：校准过程使用有限状态机管理4个采样点

详见代码：`Src/touch.c`, `Src/touch_ui.c`, `Src/touch_calib.c`
